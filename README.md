Osobní portfolio | Lucie Satrapová
Tento projekt představuje mé osobní webové portfolio, které slouží jako prezentace mých dovedností, školních i osobních projektů a mé cesty světem informačních technologií. Cílem bylo vytvořit moderní, plně responzivní a interaktivní web, který odráží mou vášeň pro frontend development a UX design.

Web je rozdělen do čtyř hlavních sekcí: Domů, Projekty, O mně a Kontakt. Design klade důraz na čistotu (whitespace), moderní typografii a jemné interaktivní prvky. Součástí je dynamická sekce "Projekty" s modálním oknem pro detailní náhledy (včetně podpory videí a galerií) a sekce "O mně", která netradičním způsobem propojuje svět IT s mým koníčkem – fitness.

Stránka je technicky postavena na čistém kódu bez použití robustních frameworků, což demonstruje pochopení základních principů webového vývoje. Obsahuje pokročilejší CSS techniky (CSS Grid, Flexbox, animace, proměnné) a vlastní JavaScript pro obsluhu interaktivity (plovoucí menu, canvas animace na pozadí, filtrování a modální okna).

Použité technologie
Projekt je postaven na standardním JAMstack základu:

HTML5: Sémantická struktura a validní kód.

CSS3:

CSS Variables (Custom Properties) pro snadnou správu barev.

CSS Grid a Flexbox pro layout (včetně složitějšího "split" designu).

Keyframe animace a transitions pro plynulý uživatelský zážitek.

Responzivní design (Media Queries) pro mobilní zařízení a tablety.

JavaScript (ES6+):

Manipulace s DOM (otevírání/zavírání modálních oken, hamburger menu).

HTML5 Canvas API (interaktivní animace částic/vln na pozadí).

Logika pro galerii obrázků (carousel) a detekci video obsahu.

Google Fonts: Písmo Inter pro moderní a čitelný vzhled.


Pro sledování návštěvnosti jsem implementovala nástroj Google Analytics 4. Do hlavičky (<head>) všech podstránek jsem vložila měřicí skript (Global Site Tag - gtag.js). Díky tomu mohu sledovat počet návštěvníků, jejich geografickou polohu a chování na stránce (např. které projekty si prohlížejí nejčastěji).
